# 高校社团学生会管理系统 python1172

## 项目概述

高校社团学生会管理系统是基于Python开发的一套在线平台，旨在帮助高校内的社团和学生组织进行高效的信息管理和活动组织。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：HTML/CSS/JavaScript（具体框架未提及）
- 开发工具：PyCharm
- Python版本：3.7

## 功能模块

### 前台功能

- 社团公告
- 用户注册
- 社团活动查看与报名
- 在线留言板

### 后台功能

- 管理员用户管理
  - 社团用户管理
  - 注册用户管理
  - 社团活动管理
  - 社团报名管理
  - 留言板管理
  - 系统管理

### 社团管理员功能

- 活动公告管理
- 社团活动管理（发布）
- 社团报名管理（审核）

### 注册用户功能

- 个人资料管理
- 社团报名结果查看

## 系统角色

- 系统管理员
- 社团管理员
- 注册用户

## 运行环境

- 操作系统：兼容Windows、Linux等
